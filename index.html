erawan-cluster (Go)

Go REST API for dynamic HAProxy config and MySQL cluster automation via Ansible.

Run

go mod tidy
make run

Health

GET /health

HAProxy Endpoints

POST   /haproxy/config
DELETE /haproxy/config
GET    /haproxy/configs
POST   /haproxy/reload

MySQL Cluster Endpoints

POST /cluster/mysql/deploy
GET  /cluster/mysql/jobs
GET  /cluster/mysql/jobs/{jobID}
POST /cluster/mysql/jobs/{jobID}/resume
POST /cluster/mysql/jobs/{jobID}/rollback
